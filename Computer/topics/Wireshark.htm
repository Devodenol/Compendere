COMPUTER
Computer > Networking > Tools >
Wireshark

Launching

  • sudo wireshark

Capture Filters

  • Capture > Capture filters to choose from predefined filters
  • net 10.11.1.0/24 using the net filter to only capture traffic on the 10.11.1.0/24 address range
  • start the capture by selecting the network interface

Display Filters

  • Analyze > Display filters to choose from predefied list
  • tcp.port == 21 to display only tcp traffic on port 21 (used by FTP)

Follow TCP Stream

  • Display the client-server communication in plaintext works on web traffic, nc (shells, mail, unencrypted connections), ftp
  • Right-click a packet > Follow > TCP Stream
