Computer > Shell >
User
hostname display hostname of the system.
passwd change password of current user
passwd username setting another user's password as root
useradd username adding a user as root
userdel username removing a user as root, use -r to remove its home folder
groups username listing grps a user is in
whoami username of the users logged in at the terminal
who list of all the users currently logged in as a user.
w display current system status, time, duration, list of users currently logged in on system and otheruser information
last who recently used the system
lastb shows all bad login attempts into the system
