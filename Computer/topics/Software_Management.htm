Software Management

Advanced Package Tool (ATP) is used to manage packages on Debain-based system

sudo apt update
update the locally cached info about the ATP packages 
sudo apt upgrade
upgrade the installed packages 
apt-cache search app_name  search cached database to find whether the application 
apt show app_name  show info about the package 
sudo apt install app_name  to install the package and its dependencies 
sudo apt remove --purge app_name  remove a package completely 

Adding Repositories

  • The servers that hold the software for particular distributions of Linux are known as repositories.
  • The repositories your system will search for software are stored in the sources.list file at /etc/apt/sources.list
  • To add a repository, just edit the sources.list file by adding the name of the repository to the list and then save the file.

Github

  • git clone github_url_of_software : copies all the data and files from that location onto your system

dkpg is the core tool to install .deb file

sudo dpkg -i filename.deb install a package
dpkg -l  output a list of all installed packages  
dpkg -L package_name  files installed and path details for a given package
