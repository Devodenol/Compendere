Recon-ng

kali@kali:~$ recon-ng run recon-ng
[recon-ng][default] > marketplace search github search marketplace for github modules
https://github.com/lanmaster53/recon-ng-marketplace/wiki/API-Keys modules with an asterisk in the K column require credentials or API keys
[r][d] > marketplace info recon/domainshosts/google_site_web gives description about the module
[r][d] > marketplace install recon/domainshosts/google_site_web install the module
[r][d] > marketplace load recon/domainshosts/google_site_web load the module
[r][d][g] > info gives info about the module like "options"
[r][d][g] > options set SOURCE churchofsatan.com change the SOURCE options from default to a website (string)
[r][d][g] > run running the module updates the "hosts" table
[r][d][g] > back to go back
[r][d] > show gives info about using "show"
[r][d] > show hosts
show the "hosts" table
the host column contains the subdomains of churchofsatan.com
[r][d] > marketplace load recon/hosts-hosts/resolve
this module resolves the IP address for a host
in SOURCE = default uses the host from the "hosts" table having no IP
[r][d][r] > run running with SOURCE = default updates the IP column of hosts table
